NanoGUI is a minimalistic cross-platform widget library for OpenGL 3+,
GLES 2/3, and Metal. It supports automatic layout generation, stateful
C++ lambdas callbacks, a variety of useful widget types

- gui library for mitsuba renderer by wenzel jakob

  #+begin_example
 cd src
 git clone https://github.com/mitsuba-renderer/nanogui
 cd nanogui
 git submodule update --init --recursive
  #+end_example
